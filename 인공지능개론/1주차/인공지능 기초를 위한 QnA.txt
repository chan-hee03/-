인공지능 기초를 위한 FAQ

1. 인공지능에서 지능에 해당하는 기능은 무엇인가?

학습, 추론, 문제해결, 패턴인식 등의 기능을 통하여 사람의 인지능력을 모방하는 부분이다.

2. 인공지능의 종류 3가지에 대해서 설명하시오 (지도학습, 반지도학습, 강화학습)

지도학습: 샘플특성과 타겟으로 학습
반지도학습: 샘플특성만으로 학습
강화학습: 보상을 통한 학습

3. 전통적인 프로그래밍 방법과 인공지능 프로그램의 차이점은 무엇인가?

전통적인 프로그래밍은 사람이 입력을 분석하여 기준(rule)을 정함
인공지능 프로그래밍은 컴퓨터가 데이터와 타깃값을 입력받아 스스로 rule을 학습함

4. 딥러닝과 머신러닝의 차이점은 무엇인가?

머신러닝은 사람이 먼저 타깃을 구분하는 데에 필요한 특성을 선출한 후 입력을 해주면 
그 데이터를 통하여 학습을 함.
딥러닝은 스스로 생각해 특성들 중에서 필요한 특성을 구분해내 바로 학습함.

5. Classification과 Regression의 주된 차이점은?

분류와 회귀
CLassification은 하나의 이산적인 값을 추측함
Regression은 계속되는 값에 대하여 추측함. 

6. 머신러닝에서 차원의 저주(curse of dimensionality)란?

데이터 학습을 위한 차원이 증가하여 학습데이터의 수보다 많아질경우 성능이 저하되는 현상
차원이 늘어날수록 학습 데이터가 없는 빈공간이 늘어나기 때문에 발생한다

7. Dimensionality Reduction는 왜 필요한가?

차원의 저주를 줄이기 위하여 필요하다.

8. Ridge와 Lasso의 공통점과 차이점? (Regularization, 규제 , Scaling)

공통점: 머신러닝 모델이 훈련세트를 과도하게 학습하지 않도록 규제하는 모델이다.
        각 특성에 곱해지는 계수를 작게 만들어 조정한다.
차이점: Ridge는 계수를 작게 만들기만 하지만 Ridge는 0으로 만들어 불필요한 변수는 제거한다

9. Overfitting vs. Underfitting

Overfitting(과대적합) : train 세트를 너무 과하게 학습하여 실제 test 세트에 대한 score가 낮은 상태
Underfitting(과소적합): train 세트와 test 세트 양쪽에서의 score가 낮은 학습상태

10. Feature Engineering과 Feature Selection의 차이점은?

Feature Engineering(특성공학)은 특성을 변형,결합하여 데이터를 확장하는 과정이지만 
Feature Selection(특성선택)은 반대로 불필요한 특성을 제거하여 데이터를 축소한다

11. 전처리(Preprocessing)의 목적과 방법? (노이즈, 이상치, 결측치)

원본 데이터 특성을 정리하고 변환하여 모델의 성능을 향상시키는 과정이다.
노이즈나 이상치를 제거하거나 결측치를 보완하고, 특성값의 스케일을 줄여 안정성을 확보하는 등의 
과정을 통하여 이루어진다

12. EDA(Explorary Data Analysis)란? 데이터의 특성 파악(분포, 상관관계)

수집한 데이터를 이용하기 전 시각화 과정을 통해 데이터의 특성을 직관적으로 확인해보는 것이다
데이터의 분포나 상하관계를 미리 알아봄으로써 데이터 이용 방향을 결정할 수 있다.

13. 회귀에서 절편과 기울기가 의미하는 바는? 딥러닝과 어떻게 연관되는가?

회귀에서 절편은 독립변수가 0일 때 예상되는 기본값이고, 기울기는 독립변수가 1이 증가 할 때마다 
종속 변수가 변하는 크기이다.
이는 딥러닝에서 편향과 가중치로 작용하여 신경망학습을 가능하게 한다.
가중치는 주어진 데이터 특성 가중치를 담당하고, 편향은 절편 역할을 담당한다

28. 결정트리에서 불순도(Impurity) – 지니 계수(Gini Index)란 무엇인가?

결정트리는 노드에서 데이터를 분할하면서 정보를 정리한다. 
이때 각 노드에서 데이터가 혼합된 정도(불순도)를 계산하여 최적의 분할을 찾는다.
지니계수는 노드에서 무작위로 데이터 샘풀을 선택했을 때 잘못 분류될 확률을 의미한다.
지니계수가 작을수록 불순도가 낮아 더 좋은 분할임을 나타낸다.

29. 앙상블이란 무엇인가?

여러개의 모델을 결합해 더 좋은 성늘을 뽑아내는 것.

30. 부트 스트랩핑(bootstraping)이란 무엇인가?

데이터 속에서 여러번 랜덤한 샘플을 뽑아내며 분석을 수행하여 통계적인 학습을 이루는 방법이다.

31. 배깅(Bagging)이란 무엇인가?

부트스트랩핑과 앙상블을 결합한 학습방법.
여러 모델들을 독립적으로 학습 -> 부트스트래핑 사용 
-> 각 모델의 결과를 평균or다수결로 종합하여 결합한다.

32. 주성분 분석(PCA) 이란 무엇인가?

Dimensionality Reduction 기법 중 하나
데이터를 시각화 할 수 있게 한다다
데이터 정규화 → 공분산 행렬 계산 → 고윳값과 고유벡터 계산 → 주성분 선택 → 데이터 변환
